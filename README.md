# Algorithm Visualizer

An interactive algorithm visualizer built with React and Tailwind CSS, featuring animated sorting algorithms, real-time controls, and an educational UI.

## Features

- Visualizes sorting algorithms step by step
- Supports Bubble Sort, Merge Sort, Selection Sort, Quick Sort, Insertion Sort and Heap Sort
- Pause and resume animations
- Adjustable animation speed
- Dynamic algorithm information panel

## Tech Stack

- React
- Tailwind CSS
- Vite

## Purpose

This project was built to explore algorithm behavior visually and improve understanding of time and space complexity through interactive learning.
